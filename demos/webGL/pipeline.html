The Graphics Pipeline

In graphics there are very many independent calculations going on to create a scene.

Often, the computer is doing the same calculations with different bits of data. For example, all the points (or vertices) that make up a model could be moved to the right place in a 3D scene, or all the pixels that make up a triangle could be coloured using the same lighting model.

It doesn't matter which calculations are done first. They could be done in any order, or they could be done all at the same time

This is how your graphics hardware works. It can perform the same calculations on thousands of different bits of data at the same time.

Today we're going to look at three parts of the process. They are:

1. Loading data into graphics buffers.

2. Moving vertices around in 3D space.

3. Calculating what colour individual pixels should be.
